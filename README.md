# 🚀 NiceAtNoon Website Clone

A modern, responsive frontend clone of the **NiceAtNoon** website built to practice advanced UI development, smooth scrolling, and animation techniques.

## 🌐 Overview

This project is a frontend recreation of the NiceAtNoon website. The goal was to improve my skills in responsive web design, modern animations, and interactive user interfaces using industry-standard frontend technologies.

> **Disclaimer:** This project was created for educational and portfolio purposes only. All design credits belong to the original NiceAtNoon website.

---

## ✨ Features

* 🎨 Pixel-perfect UI inspired by the original website
* 📱 Fully Responsive Design
* 🚀 Smooth Scrolling using **Locomotive Scroll**
* 🎬 Scroll-based Animations with **GSAP** & **ScrollTrigger**
* 🖱️ Interactive Hover Effects
* ⚡ Modern Landing Page Design
* 🧩 Clean and Organized Code Structure

---

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS
* JavaScript (ES6)
* GSAP
* ScrollTrigger
* Locomotive Scroll
